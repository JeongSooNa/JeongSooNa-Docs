# Class 3 - Python 자료형

### 변수 선언과 정의
- 선언 : 새로운 변수 생성
- 정의 : 변수의 값 지정

```py
# 파이썬에서는 변수 선언과 함께 정의가 동시에 이뤄진다.
# 변수명은 영문자 혹은 언더바(_)로 시작
a = 1

print(a)
```

```py
a = 1
a = 10
print(a)
```

### 타입
- 변수의 종류, 자료형

```py
# int(integer)
a = 1

# float
b = 1.2

# str(string, character)
c = "Python"

# bool(Boolean, True/False)
d = True

# NoneType

e = None

print("a의 타입은 : ",type(a))
print("b의 타입은 : ",type(b))
print("c의 타입은 : ",type(c))
print("d의 타입은 : ",type(d))
print("e의 타입은 : ",type(e))
```
