### Github
- 개발자들을 위한 오픈소스 코드 공유 플랫폼
- 개인저장, 협업, 코드리뷰, 이슈 트래킹 등 다양한 용도로 사용되고 있다.

- 용어
    - Repository : 저장소
    - Commit : 본인의 환경에 임시 저장
    - Push : Commit 내용을 저장소에 저장
    - Pull : 저장소를 내 환경에 불러오기
