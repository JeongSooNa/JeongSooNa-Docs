# Class 9 - R Statistical Computing

### Basic

- 전산통계 개요 & 통계적 컴퓨팅의 필요성

- 확률분포의 모의실험 (Random number generation)

- 난수 발생기, 의사난수, 분포별 샘플링

- 수치적 방법 (Numerical Methods)

- 수치적 적분, 최적화, 루트 찾기 알고리즘

- 몬테카를로 시뮬레이션

- 퍼머테이션 테스트 (Permutation Test)

- Variational Inference (선택적으로 다룸)
