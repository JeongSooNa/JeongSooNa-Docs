# Class 0 - R 기초 프로그래밍

R은 통계나 유전학 연구 분야에서 주로 사용되는 오픈소스 프로그래밍 언어로, 복잡한 수식이나 통계 알고리즘을 간단히 구현해 사용할 수 있다.

### R 설치
